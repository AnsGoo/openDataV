# 符号文本（`Symbol1`）

> 展示一个符号文本，主要用于页面符号展示

## 版本信息

|组件版本|平台版本|说明|
|:---:|:---:|:---:|
|1.0|{{ $frontmatter.platform }}|可用|

## 属性

### 基础配置

|配置项|作用|说明|
|:---:|:---:|:---:|
|`符号`| 符号|具体符号可参考，编辑器上面的图标|


## 样式

### 字体设置

|配置项|作用|说明|
|:---:|:---:|:---:|
|`颜色`| 文字颜色，支持16进制和RGBA颜色表示方法||
|`字体大小`|字体大小||
